Getting information in – every source becomes tokens in the same window

Claude can only work with what's inside its context window, but there are several ways to get external information in. You can attach files manually in Claude.ai, let Claude Code read files directly from your machine, or use connectorsMCP (Model Context Protocol)A standard that lets Claude reach out to external data sources like SharePoint, Slack, email, and databases. Called "connectors" in the Claude.ai interface.Think of it as giving Claude permission to look things up on your behalf. to pull in live data from systems like SharePoint, Slack, and databases. No matter which method you use, the information always arrives the same way – as tokens added to the context window.
